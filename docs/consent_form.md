## 一、研究目的

本研究旨在使用智能手机视频和人工智能技术，分析弹跳动作中的关键技术因素，探索哪些动作特征对跳跃高度影响最大。研究结果将为体育训练提供科学依据。

## 二、研究内容

如果您同意参加本研究，您将：

1. 提供基本个人信息（年龄、身高、体重等）
2. 接受简单的体能测试（立定跳远）
3. 在指定地点进行5次弹跳摸高测试
4. 测试过程将被智能手机视频录制

**预计所需时间**：约15-20分钟

## 三、潜在风险

本研究仅涉及常规的弹跳运动，与正常体育课活动强度相当。潜在风险包括：
- 轻微的肌肉疲劳
- 极小概率的运动损伤（如扭伤）

**安全措施**：
- 测试前进行充分热身
- 提供安全的测试场地
- 有基本急救措施准备

## 四、隐私保护

1. 您的个人信息将严格保密，仅用于本研究
2. 数据将进行匿名化处理，使用编号代替姓名
3. 视频数据仅用于提取动作特征，不会公开发布
4. 研究结束后，原始视频将被删除或脱敏保存
5. 研究成果发表时不会出现可识别个人身份的信息

## 五、自愿参与

1. 参与本研究完全自愿
2. 您有权在任何时候退出研究，无需说明理由
3. 退出研究不会对您产生任何不利影响
4. 如退出，您的数据将被删除（如尚未匿名化处理）

## 六、研究收益

1. 您将获得一份个人弹跳动作分析报告
2. 报告将包含您的技术特点和改进建议
3. 您的参与将为体育科学研究做出贡献
