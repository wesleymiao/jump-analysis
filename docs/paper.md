## 摘要

**背景**: 弹跳能力是篮球、排球等运动的核心体能素质，但传统的弹跳分析依赖专业运动生物力学实验室，成本高昂且难以普及。

**目的**: 本研究旨在开发一套低成本、可解释的弹跳动作分析系统，利用智能手机视频和机器学习技术，量化分析影响弹跳高度的关键动作特征，并为个体提供针对性的技术改进建议。

**方法**: 招募30名高中生志愿者，使用智能手机以60fps帧率从侧面拍摄弹跳摸高视频。通过MediaPipe姿态估计提取33个骨骼关键点，计算8项生物力学特征（膝关节蹬伸角速度、手臂摆动幅度等）。采用XGBoost回归模型建模，并使用SHAP方法解释各特征对弹跳高度的贡献。为控制个体差异，采用混合效应模型和个体内重复测量设计。

## 1. 引言

### 1.1 研究背景

弹跳能力是衡量下肢爆发力的重要指标，在篮球、排球、跳高等运动中起着决定性作用。传统的弹跳分析需要专业的运动生物力学实验室，配备高速摄像机、三维动作捕捉系统、测力板等设备，成本高昂，难以在普通学校和训练场所推广。

近年来，计算机视觉技术的快速发展为低成本运动分析提供了新的可能。以MediaPipe为代表的姿态估计算法可以从普通视频中实时提取人体骨骼关键点，为基于智能手机的运动分析奠定了技术基础。

### 1.2 研究问题

本研究旨在回答以下问题：

1. **技术可行性**: 使用智能手机视频和开源姿态估计算法，能否准确提取弹跳动作的生物力学特征？

2. **关键因素识别**: 在控制个体身体素质差异的条件下，哪些动作技术特征对弹跳高度影响最大？

3. **可解释性**: 如何让机器学习模型的预测结果具有可解释性，从而为运动员提供具体的技术改进建议？

### 1.3 研究意义

**理论意义**:
- 将可解释机器学习方法引入运动生物力学领域
- 探索低成本运动分析的可行性边界

**实践意义**:
- 为学校体育教学提供科学化的弹跳训练指导工具
- 为运动员提供个性化的技术诊断和改进建议

### 1.4 研究创新点

1. **低成本数据采集**: 仅需智能手机，无需专业设备
2. **可解释的AI模型**: 使用SHAP方法量化每个技术因素的贡献
3. **混合效应设计**: 通过个体内重复测量控制个体差异
4. **端到端系统**: 从视频输入到个性化建议输出的完整解决方案

## 2. 文献综述

### 2.1 弹跳生物力学研究现状

#### 2.1.1 弹跳的生理学基础

垂直弹跳涉及下肢三个主要关节的协调运动：髋关节、膝关节和踝关节。根据Linthorne (2001)的研究，弹跳高度主要取决于离地瞬间的垂直速度，符合抛体运动规律：

$$h = \frac{v_0^2}{2g}$$

其中 $h$ 为跳跃高度，$v_0$ 为离地速度，$g$ 为重力加速度。

#### 2.1.2 影响弹跳高度的因素

根据现有文献，影响弹跳高度的因素可分为两类：

**不可改变因素（个体属性）**:
- 肌纤维类型比例（快肌/慢肌）
- 肢体长度比例
- 神经肌肉协调能力

**可改变因素（技术动作）**:
- 下蹲深度与蹬伸速度
- 手臂摆动幅度与时机
- 躯干姿态控制

| 研究者 | 年份 | 主要发现 |
|--------|------|----------|
| Harman et al. | 1990 | 手臂摆动可提高跳跃高度约10% |
| Bobbert et al. | 1996 | 最佳下蹲深度因个体而异 |
| Feltner et al. | 1999 | 手臂摆动的时机与下肢协调至关重要 |

### 2.2 计算机视觉在运动分析中的应用

#### 2.2.1 姿态估计技术发展

### 2.3 可解释机器学习

#### 2.3.1 XGBoost算法

XGBoost (Extreme Gradient Boosting) 是一种基于梯度提升的集成学习算法，由Chen和Guestrin于2016年提出。其核心思想是通过迭代训练多棵决策树，每棵新树专注于纠正前面所有树的预测误差。

模型预测公式：
$$\hat{y}_i = \sum_{k=1}^{K} f_k(x_i)$$

其中 $K$ 为树的数量，$f_k$ 为第 $k$ 棵树。

#### 2.3.2 SHAP解释方法

SHAP (SHapley Additive exPlanations) 基于博弈论中的Shapley值，将模型预测分解为各特征的贡献：

$$\hat{y} = \phi_0 + \sum_{j=1}^{M} \phi_j$$

其中 $\phi_0$ 为基准预测（所有样本的平均值），$\phi_j$ 为特征 $j$ 的SHAP值。

### 2.4 研究空白与本研究定位

现有研究的不足：
1. 专业设备成本高，难以推广
2. 机器学习模型多为"黑箱"，缺乏可解释性
3. 较少考虑个体差异的控制

本研究的定位：填补上述空白，开发一套低成本、可解释、考虑个体差异的弹跳分析系统。

#### 3.3.2 生物力学特征计算

本研究提取8项核心生物力学特征：

| 编号 | 特征名称 | 计算方法 | 物理意义 |
|------|----------|----------|----------|
| F1 | 最小膝关节角度 | 下蹲时髋-膝-踝三点夹角最小值 | 下蹲深度 |
| F2 | 膝关节蹬伸角速度 | 膝关节角度变化率的最大值 | 蹬地爆发力 |
| F3 | 髋关节角度变化 | 离地时髋角 - 下蹲时髋角 | 伸髋幅度 |
| F4 | 手臂摆动幅度 | 手腕垂直位移最大值 | 上肢助力 |
| F5 | 手臂摆动时机 | 手臂最高速度时刻 - 离地时刻 | 协调性 |
| F6 | 蹬伸时间 | 从最低点到离地的时间 | 发力节奏 |
| F7 | 躯干前倾角 | 离地时肩-髋连线与垂直线夹角 | 身体姿态 |
| F8 | 离地瞬间速度 | 离地时刻重心垂直速度 | 起跳速度 |

### 3.4 建模方法

#### 3.4.1 控制个体差异的策略

本研究采用三种策略控制个体差异：

**策略1：个体内重复测量**
- 每人跳5次，分析同一个体不同跳跃间的变化
- 消除个体固有特征（身高、力量）的影响

**策略2：协变量控制**
- 纳入个体属性作为控制变量
- SHAP分别显示技术因素和体质因素的贡献

**策略3：混合效应模型**
```
跳跃高度 = 固定效应（技术特征）+ 随机效应（个体差异）+ 误差

yᵢⱼ = β₀ + β₁x₁ᵢⱼ + β₂x₂ᵢⱼ + ... + uᵢ + εᵢⱼ

其中：
- yᵢⱼ: 第i个人第j次跳跃的高度
- xᵢⱼ: 技术特征（每次跳跃不同）
- uᵢ: 个体随机效应（每人一个基础能力值）
- εᵢⱼ: 随机误差
```

#### 3.4.2 XGBoost模型

**模型参数**:
```python
XGBRegressor(
    n_estimators=100,      # 100棵树
    max_depth=4,           # 最大深度4层
    learning_rate=0.1,     # 学习率
    min_child_weight=3,    # 最小叶子权重
    subsample=0.8,         # 样本采样比例
    colsample_bytree=0.8,  # 特征采样比例
    random_state=42        # 随机种子
)
```

**交叉验证**:
- 采用5折交叉验证评估模型性能
- 评估指标：R²、RMSE、MAE

#### 3.4.3 SHAP分析

**全局解释**:
- 特征重要性排名：各特征SHAP值绝对值的均值
- 特征影响方向：正值表示提高跳跃高度，负值表示降低

**局部解释**:
- 对每个样本计算各特征的SHAP值
- 生成个性化分析报告

### 7.3 研究局限性

1. **样本量限制**: 30人样本相对较小，结果的泛化性需进一步验证

2. **设备精度**: 智能手机视频的精度低于专业运动捕捉系统

3. **2D分析局限**: 侧面视频仅能获取矢状面的运动信息，无法分析冠状面和水平面的动作

4. **个体差异控制**: 虽然采用了混合效应模型，但仍可能存在未测量的混杂因素

5. **因果推断限制**: 观察性研究难以建立严格的因果关系

### 7.4 改进方向

1. 扩大样本量，纳入不同运动水平的群体
2. 结合多角度视频或3D姿态估计
3. 增加力板数据验证地面反作用力
4. 开展干预研究验证训练效果
